# Proyecto-Final-PW
📚 Librería Áurea — Portal Académico de Gestión Bibliográfica

Autor: Michael Sánchez
Matrícula: 2024-0063

🧾 Descripción

Librería Áurea es una aplicación web desarrollada como proyecto final, cuyo objetivo es gestionar y consultar un catálogo de libros y autores mediante el uso combinado de HTML, CSS, JavaScript, PHP y MySQL.

El sistema permite a los usuarios explorar el catálogo bibliográfico de forma intuitiva, mientras que el administrador cuenta con un panel privado para la gestión completa de la información.

🚀 Funcionalidades principales
🌐 Sitio público
Consulta de libros con filtros y ordenamiento
Consulta de autores
Buscador en tiempo real
Formulario de contacto con validaciones
Interfaz moderna con modo claro/oscuro
🔐 Panel de administración
Sistema de autenticación (login seguro)
Panel privado para gestión del sistema
CRUD completo de:
Libros
Autores
Gestión de mensajes enviados desde el formulario de contacto
Exportación de mensajes en formato CSV
Dashboard con estadísticas
⚙️ Creación de administrador (importante)

⚠️ Seguridad

Después de crear el usuario administrador:

❗ Debes eliminar el archivo setup_admin.php del servidor

Esto evita que cualquier persona pueda crear nuevos administradores.

🎯 Objetivo del proyecto

Este proyecto fue desarrollado con fines académicos para poner en práctica el desarrollo de aplicaciones web completas, integrando frontend y backend, manejo de bases de datos y buenas prácticas de programación y seguridad.

📌 Nota

El sistema está orientado a la consulta y gestión de catálogo bibliográfico, por lo que no incluye módulo de compras, ya que no formaba parte de los requerimientos del proyecto.
